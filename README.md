## AI Agent Bot
ai agent that run in esp32


### Run
1. clone this repo
2. install arduino ide
3. install esp32 board in arduino ide
4. open this project in arduino ide
5. connect esp32 to computer
6. upload sketch to esp32
7. open web ui from ip in serial monitor
8. setup wifi and bot token
9. setup ai api key and model
10. save & restart esp32
11. top open web ui again open new ip show in serial monitor
